# Preferred Tools & Processes

## Leadership & Strategy

- **Methodology:** Lean Six Sigma (DMAIC), OKRs for goal setting.
- **Documentation:** Start with a "Problem Statement" before suggesting solutions.
- **Meeting Style:** Amazon-style "Silent Read" memos preferred over slide decks.

## Project Management

- **Framework:** Agile/Scrum (2-week sprints).
- **Tracking:** User Stories should always have "Acceptance Criteria."

## Development (If Applicable)

- **Frontend:** SvelteKit (TypeScript).
- **Backend:** Python 3.11+ (FastAPI).
- **AI:** LangChain or native API calls preferred.
